# Assembler用户手册

*基本使用指南*


## 功能说明

### 主要解决问题

主要解决两大类问题：

- 数据可视化
	* 之前的模块完成的是原始网站数据提取与整理
	* 但是对于用户来说无法直接理解这些数据
	* 本模块功能是将数据与样式进行整合，从而方便普通用户、残障人士在内的各类用户使用
	
## 使用方式
* 首先以测试身份登陆Production Server，命令类似于：```ssh -i esd_rsa your_name@prod_server```
* 使用```cd```命令进入目标网站所在的目录
* 运行```lab run assembler```执行即可
