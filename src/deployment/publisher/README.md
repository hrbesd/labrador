## 版本服务器的目录布局

### 约定

- 用户名是 `updater`
- 产品根目录是 `~/labrador`

## 如何实现

先在本地建立好目录结构，然后把要复制的 `butts` 文件用符号链接，使用 `rsync` 把符号链接当作文件复制。然后在 `bin` 目录里给 `butts` 里的文件建立符号链接，把符号链接按照符号链接复制。
